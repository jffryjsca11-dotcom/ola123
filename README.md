## Descripción del Proyecto
Este proyecto consiste en una aplicación web basada en **Flask** que actúa como servidor de visualización para datos de dimensiones  `(X, Y)`  

La interfaz gráfica procesa los valores del sensor y los mapea directamente sobre una matriz  de **5x5**, iluminando  la celda correspondiente.
